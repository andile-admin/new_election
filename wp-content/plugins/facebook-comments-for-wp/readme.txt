== Description ==

This plugin adds Facebook’s Comments Plugin to your blog, instead of natural WordPress Comments Form.
If you want to see a demo, you can check [this page](http://erayalakese.com/facebook-comments-for-wp/ "Eray Alakese").

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php fcfw_show(); ?>` in your template, instead of `comments_template()`
1. Place `<?php fcfw_comment_count(); ?>` in your template, if you want to print comments count for recently page.
